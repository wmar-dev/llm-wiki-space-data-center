# Edge Computing in Low-Earth Orbit — What Could Possibly Go Wrong?

**Authors**: Tobias Pfandzelter, David Bermbach
**Source**: arxiv:2302.08952
**URL**: https://arxiv.org/abs/2302.08952

## Abstract

Addresses software adaptation needs for LEO edge computing systems, focusing on handling satellite mobility and failure patterns. Presents a taxonomy of failures that may occur in LEO edge computing and derives considerations for software systems. Failure patterns include effects of radiation on compute hardware in space.
